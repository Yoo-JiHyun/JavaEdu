JAVA 데이터처리

	- 데이터 출력 메소드
		System.out.print() : 매개변수의 인자로 전달된 데이터를 콘솔에 출력
		System.out.printIn() : 매개변수의 인자로 전달된 데이터를 콘솔에 출력 후 줄바꿈
		System.out.printf() : 매개변수의 인자로 전달된 형식으로 데이터를 콘솔에 출력
		
	- 변수
		메모리에 데이터를 저장할 공간을 만드는 기능과 메모리 공간의 주소를 저장하는 기능을 가짐
		
	- 변수 선언
		자료형 타입을 정의하고 식별할 수 있는 이름을 정의
		
	- 데이터 타입
		논리값을 저장하는 타입 : Boolean
		문자값을 저장하는 타입 : char
		정수값을 저장하는 타입 : byte, short, int, long
		실수값을 저장하는 타입 : float, double
		참조값을 저장하는 타입 : String
		
	- 데이터 자료형의 형 변환
		범위가 작은 변수가 넓은 변수로 변경되는 경우는 자동 형변환 처리
		넓은 범위의 자료형이 작은 범위의 자료형으로 변경해야 할 경우 명시적으로 형변환 연산자를 이용해서 형변환 처리
