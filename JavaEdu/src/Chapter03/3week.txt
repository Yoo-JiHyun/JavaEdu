JAVA 연산자
	- 연산자
		연산의 형태를 정의하는 기호를 의미
		
	- 피연산자
		연산의 대상이 되는 변수 또는 상수, 리터럴
		
	- JAVA에서 사용되는 연산자의 종류
		산술 / 부호 / 문자열 / 대입 / 증감 / 비교 / 논리 / 조건 / 비트 / 시프트
		
	- 증감연산자
		변수의 값을 1씩 증가 , 감소시키는 연산자
		
	- 비교연산자
		크다, 작다, 같다 등의 비교 연산을 의미하며 비교 결과를 Boolean 타입의 논리값으로 반환
		
	- 논리 연산자
		논리곱, 논리합, 논리 부정 등의 연산을 수행하고 Boolean 타입의 논리값을 반환
		
	- 대입 연산자와 복합대입 연산자
		대입 연산을 오른쪽에서 왼쪽으로 수행 , 복합대입 연산자는 연산과 대입을 복합적으로 수행
		
	- 삼항 연산자
		(조건식 ) ? X : Y 형식의 연산으로 조건식의 결과에 따라 X와 Y를 반환
		
	- 연산의 결과 값
		연산을 위해서는 자료형 타입을 일치시킨 후 , 일치 시킨 자료형 타입의 연산결과를 반환
